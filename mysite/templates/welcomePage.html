# Introduction

This is a tiny project for ML beginers to see how is the learning process of a neural network on a dataset by visualizing the loss plot and the parameters in a real-time way. Currently, I fix the model structure and the dataset (the MNIST). In the future, it would allow user to control the model structure via the interface and upload customized dataset.

{% load staticfiles %}

# Pre-requests

Python3, django(version2.0.1), matplotlib, Numpy, Scipy
matplotlib needs _tkinter modules for some linux(ubuntu) users. use "apt install python3-tk" to install the tkinter for python3
